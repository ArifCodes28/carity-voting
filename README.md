# Charity Voting Smart Contract (Soroban - Stellar)

## 📌 Project Description

Charity Voting is a decentralized smart contract built using Soroban on the Stellar blockchain.
It allows users to vote for their preferred charities in a transparent, secure, and tamper-proof way.

This project demonstrates how blockchain can be used to improve trust and fairness in donation and charity selection systems.

---

## 🚀 What It Does

* Users can vote for a charity using their wallet address
* Each user is allowed to vote only once
* Votes are stored on-chain (immutable and transparent)
* Anyone can view the total votes for each charity

---

## ✨ Features

* 🔐 **Secure Voting** – Uses blockchain authentication
* 🚫 **No Double Voting** – Each address can vote only once
* 📊 **Transparent Results** – Votes are publicly verifiable
* ⚡ **Fast & Low Cost** – Built on Stellar Soroban
* 🧩 **Simple Structure** – Easy to extend (e.g., donation integration)

---

## 🛠️ Tech Stack

* Rust (Soroban SDK)

* Soroban Smart Contracts
